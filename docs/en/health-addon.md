# Health add-on

[Docs index](../README.md) · [Español](../es/health-addon.md)

The health add-on adds optional tracking for body metrics and routines.

## Quick path

1. Enable the health add-on
2. Enable the modules you want
3. Complete the body profile if you use weight tracking
4. Open the health view

## Available modules

- weight and BMI
- blood pressure
- medications

## Weight tracking

The weight module supports:

- body profile setup
- weight entries
- BMI context
- progress and trend views
- CSV export

## Blood pressure

The blood pressure module supports:

- systolic/diastolic readings
- pulse logging
- reading history

## Medications

The medications module supports:

- medication records
- inventory
- intake logs
- schedule notes

## Important note

This feature is informational and organizational. It is not medical advice.
