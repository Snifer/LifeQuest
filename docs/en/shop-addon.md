# Shop add-on

[Docs index](../README.md) · [Español](../es/shop-addon.md)

The shop add-on lets you convert progress into redeemable rewards using coins.

## Quick path

1. Enable coin rewards
2. Enable the shop add-on
3. Open the Hero Shop
4. Add or redeem rewards

## What it does

The shop add-on gives you:

- a reward catalog
- coin balance tracking
- reward redemption
- reward history through the coin ledger

## Dependencies

The shop depends on:

- `coinsEnabled`
- `shopEnabled`

If coin rewards are disabled, the shop UI is not enough on its own.

## Default reward ideas

Examples include:

- coffee
- media time
- food rewards
- social plans
- recovery time

## Best practices

- keep rewards meaningful but reachable
- avoid using only expensive rewards
- mix small and medium rewards
- use rewards you actually care about
